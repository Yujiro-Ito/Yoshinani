# 4-C 固有名詞辞書 / 文体プロンプト

## 概要

変換品質を上げる磨き。**固有名詞辞書**（ユーザー語彙）と、zenz への**文体プロンプト**
（です・ます／だ・である等）を設定で持たせる。

## スコープ

- 設定（`settings.json`、2-A/1-C で導入済みの Settings を拡張）に辞書・文体を追加。
- 変換要求（3-A/3-C）に辞書・文体ヒントを載せる。

## 前提

- Phase 3 完了。Settings 基盤（[[phase1-trigger-commit]] の分離）を流用。

|--|--|

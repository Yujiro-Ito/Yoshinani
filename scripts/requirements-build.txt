# よしなに(Yoshinani) — ビルドツール（pip 経由で入れる CLI）
# 用途: C++ ビルドの駆動。CMake が構成を生成し Ninja がコンパイルを実行する。

# バージョン方針: VS 同梱の壊れた CMake 3.20 を避けるため下限を切る。
#   固定したい場合は `cmake==4.3.2` のように書き換える。
